# OVS Project Signal Dashboard

**Course:** COS349 — Project and Portfolio IV  
**Milestone:** Week 1 Proof of Concept  
**Student Project:** OVS Project Signal Dashboard

## Project Summary

OVS Project Signal Dashboard is a proof-of-concept productivity and project-health dashboard for creative and technical projects. It loads local project data, summarizes key metrics, displays project cards, and allows filtering by project status.

## Week 1 Scope

This first iteration focuses on proving the application direction:

- React/Vite app runs locally.
- Project data is loaded from JSON.
- Dashboard metrics summarize progress, blockers, and active projects.
- Project cards display project details and progress bars.
- Status filter demonstrates basic interface interaction.
- Folder structure supports future milestones, screenshots, reflections, and code review notes.

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Vitest
- Local JSON data
